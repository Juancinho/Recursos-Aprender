# Recursos audiovisuales y búsquedas

Algunos temas se aprenden mejor viendo a alguien usar la herramienta. Como los vídeos cambian y envejecen rápido, esta nota mezcla enlaces estables con búsquedas recomendadas.

## Docker

- Docker official YouTube: https://www.youtube.com/@DockerIo
- Búsqueda: `Docker Compose networking volumes tutorial`
- Mira: puertos, volúmenes, compose, logs.

## Git y patch

- Git videos oficiales: https://git-scm.com/videos
- Búsqueda: `git diff git apply patch tutorial`
- Mira: cómo leer hunks, no solo comandos.

## RAG/Qdrant

- Qdrant Learn: https://qdrant.tech/documentation/tutorials/
- Búsqueda: `Qdrant hybrid search BM25 dense retrieval`
- Mira: collections, payload, filters, hybrid search.

## OpenWebUI

- OpenWebUI docs tutorials: https://docs.openwebui.com/tutorials/
- Búsqueda: `Open WebUI Qdrant RAG Docker`
- Mira: variables, vector database, provider configuration.

## vLLM/LiteLLM/Qwen

- vLLM docs: https://docs.vllm.ai/
- LiteLLM docs: https://docs.litellm.ai/
- Búsqueda: `vLLM OpenAI compatible server LiteLLM proxy`
- Mira: endpoint, streaming, model name, logs.

## Wireshark/CAN

- Wireshark docs: https://www.wireshark.org/docs/
- Búsqueda: `Wireshark CAN UDS DoIP tutorial`
- Mira: packet list, packet details, filters.

## Transformers/atención

- Illustrated Transformer: https://jalammar.github.io/illustrated-transformer/
- Hugging Face Course: https://huggingface.co/learn/llm-course/chapter1/1
- Búsqueda: `scaled dot product attention from scratch pytorch`
- Mira: Q, K, V, softmax por filas, multi-head.

## Regla al ver vídeos

- [ ] Pausa y reproduce el comando.
- [ ] Escribe qué versión usa el vídeo.
- [ ] Contrasta con documentación oficial.
- [ ] No copies configuraciones viejas sin revisar.
